  2. Playing around with the breast cancer dataset

    Tue 11 July 2017
    By Karanveer Singh

    The Breast Cancer Dataset is a collection of features computed from breast mass of 569 patients. The dataset contains 30 attributes of patients - and a class label telling whether the tumour is malignant or benign. There are a total of 357 benign objects and 212 malignant objects. I decided to use this dataset for my first blog post on machine learning, since its a very straightforward dataset with no missing values and all variables being real valued (no categorical variables). This blog post will help people new to the whole scikit-learn and machine learning world.
